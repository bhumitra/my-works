1. Pre requisits--
 Must have a windows pc. This game will only run on windows.

2. You may get warning that file may harm your computer. 
-- Disable smartscreen filter under settings of your pc
-- Set user account control to lowest value under settings of your pc
-- run exe file as an admin

3. Run BatmanArkhamAsylum.exe to play the game

4. configure game window...
---right click on top left of the window that opens when you run the game.
go to--
rightclick >> properties>> layout >> 

under set window size set the height and width as given below.
width -- 148
height -- 36

5. Follow the on screen instructions and enjoy the game......
